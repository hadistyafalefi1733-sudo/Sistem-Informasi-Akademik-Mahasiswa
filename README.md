# Sistem Informasi Akademik Mahasiswa

## Deskripsi

Sistem Informasi Akademik Mahasiswa merupakan aplikasi berbasis Python yang dikembangkan sebagai Tugas Besar Mata Kuliah Pemrograman Lanjut di Universitas Serambi Mekkah.

Aplikasi ini digunakan untuk membantu pengelolaan data akademik mahasiswa, mulai dari input data mahasiswa, input nilai mata kuliah, perhitungan IPK berbobot SKS, perangkingan mahasiswa, statistik kelas, hingga pencetakan rapor.

---

## Fitur Program

- Input data mahasiswa
- Input nilai mata kuliah
- Konversi nilai huruf
- Perhitungan rata-rata nilai
- Perhitungan IPK berbobot SKS
- Ranking mahasiswa
- Statistik kelas
- Cetak rapor
- Rekap kelas
- Pencarian data mahasiswa
- Edit data mahasiswa
- Hapus data mahasiswa
- Validasi input

---

## Teknologi yang Digunakan

- Python 3
- Modul `statistics`

---

## Cara Menjalankan Program

1. Pastikan Python 3 sudah terinstal.
2. Download atau clone repository ini.
3. Buka terminal atau Command Prompt.
4. Jalankan program dengan perintah:

```bash
python akademik.py
```

5. Pilih menu yang tersedia sesuai kebutuhan.

Validasi Input: 
Program memiliki beberapa validasi, yaitu: NIM harus berupa angka. Angkatan harus berupa angka. Nilai harus berada pada rentang 0–100. SKS harus lebih besar dari 0. Jumlah mata kuliah minimal 5. Apabila pengguna memasukkan data yang tidak sesuai, program akan menampilkan pesan kesalahan dan meminta pengguna mengulangi input.

Contoh Alur Penggunaan:
Jalankan program. Input data mahasiswa. Input nilai mata kuliah. Lihat data mahasiswa. Tampilkan ranking. Cetak rapor. Keluar dari program.
