# 🎓 Sistem Informasi Akademik Mahasiswa

Sistem Informasi Akademik Mahasiswa merupakan aplikasi berbasis **Python** yang dikembangkan sebagai **Tugas Besar Mata Kuliah Pemrograman Lanjut** di Universitas Serambi Mekkah.

Aplikasi ini bertujuan membantu pengelolaan data akademik mahasiswa, mulai dari input data mahasiswa, pengolahan nilai mata kuliah, perhitungan IPK berbobot SKS, penentuan ranking mahasiswa, hingga pencetakan rapor dan rekap kelas.

---

## ✨ Fitur Program

- Input data mahasiswa
- Input nilai mata kuliah beserta SKS
- Konversi nilai angka ke nilai huruf
- Perhitungan rata-rata nilai
- Perhitungan IPK berbobot SKS
- Ranking mahasiswa berdasarkan IPK
- Statistik kelas
- Cetak rapor mahasiswa
- Rekap kelas
- Pencarian data mahasiswa
- Edit data mahasiswa
- Hapus data mahasiswa
- Validasi input

---

## 🛠 Teknologi yang Digunakan

- Python 3
- Modul `statistics`
- List
- Dictionary
- Function
- Looping (`for` dan `while`)
- Percabangan (`if-elif-else`)
- Sorting (`sort()`)

## ▶ Cara Menjalankan Program

### Persyaratan

- Python 3.x telah terinstal.

### Langkah Menjalankan

1. Clone atau download repository ini.
2. Buka folder proyek menggunakan Visual Studio Code atau terminal.
3. Jalankan program dengan perintah:

```bash
python akademik.py
```

4. Pilih menu sesuai kebutuhan.
