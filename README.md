# Sistem Informasi Akademik Mahasiswa

## Tugas Besar Pemrograman Lanjut

Program Studi Teknik Komputer

---

## Deskripsi

Sistem Informasi Akademik Mahasiswa merupakan aplikasi berbasis Python yang digunakan untuk mengelola data akademik mahasiswa secara otomatis.

---

## Fitur

- Input Data Mahasiswa
- Input Nilai Mata Kuliah
- Konversi Grade
- Hitung IPK
- Ranking Mahasiswa
- Statistik Kelas
- Cetak Rapor
- Rekap Kelas
- Cari Mahasiswa
- Edit Mahasiswa
- Hapus Mahasiswa

---

## Teknologi

- Python 3
- Statistics Module
- Visual Studio Code
